# Kondate Proto v2

レシピとレシピセットを「メニュー表」として管理し、献立から選んで回すローカルWebアプリ。
ブラウザで `index.html` を開いて使う。

## コンセプト

- **プール式**: レシピを予定ではなく献立（プール）にする。迷っても詰まない
- **二段階選択**: セットを選ぶ → 今日の料理を選ぶ
- **Menu**: Privateのレシピ帳 / Pablicのレシピカタログ、それぞれにレシピとセットがある
- **スマホファースト**: メイン利用はスマホ

## 画面構成

| # | 画面 | 説明 |
|---|------|------|
| 0 | オンボーディング | 初回のみ。使い方を説明 |
| 1 | 献立（メイン） | 今の献立をカード横スクロールで表示 |
| 1.5 | セット選択 | セットを選ぶ画面 |
| 2 | セット作成 | 新しいセットを作る |
| 3 | レシピ登録 | レシピを新規追加 |
| 4 | レシピ帳 | 自分のレシピ/セット管理（タブ切替） |
| 5 | 買い物リスト | セットから自動生成 |
| 6 | レシピカタログ | 公開レシピ/セットを探す（タブ切替） |

## デザイン方針

- **トーン**: シンプルで自然。ユーザーの感覚・体験に近い自然な言葉選び
- **カラー**: クリーム背景、ダークブラウンテキスト、オレンジアクセント
- **週を意識させない**: 「今週」ではなく「今の献立」
