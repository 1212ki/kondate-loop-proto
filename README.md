# MenuLooper

1週間の晩ご飯を入力で決め、食材リストを自動生成するローカルツールです。ブラウザで `index.html` を開いて使います。

## 起動方法
- `tools/menu-looper-proto/index.html` をブラウザで開く
- 献立ページがデフォルトで開きます（1食に複数品を追加可能）
- レシピの追加は献立画面の入力 or 「レシピDBを開く」から行う

## データ保存
- 入力内容はローカルストレージに保存されます
- 週ごとに保存されるため、週を切り替えると過去の記録が復元されます
- レシピは基準人数（1人前 or 2人前）を登録します
- 献立画面で料理名を入力し、品目を追加（候補選択 or 新規登録）
- 新規登録はそのままモーダルで詳細を入力して保存します
- 日ごとに「今日は何人前」を入力できます
- 食材リストは人数に合わせて自動生成されます
- レシピDBでは食材名・数量・グラムを登録できます（URLは任意）
